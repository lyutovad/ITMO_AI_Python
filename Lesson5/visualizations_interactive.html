Line plot

  • Подходит для показа изменения показателя по времени или по упорядоченной шкале.
  • Хорошо отвечает на вопросы: растёт ли показатель, падает ли он, есть ли сезонность, резкие скачки.
  • Если категорий много и они не упорядочены, line plot часто не лучший выбор.
import matplotlib.pyplot as plt

months = ['Jan', 'Feb', 'Mar', 'Apr']
sales = [120, 135, 150, 142]

plt.plot(months, sales)
plt.title('Продажи по месяцам')
plt.xlabel('Месяц')
plt.ylabel('Продажи')
plt.show()
Типичный случай: время, даты, шаги обучения, номер эпохи, динамика температуры, курса или продаж.

Bar plot

  • Подходит для сравнения категорий между собой.
  • Обычно отвечает на вопросы: у какой группы значение больше, у какой меньше.
  • Если важна именно форма распределения, лучше взять histogram, а не bar plot.
cities = ['Москва', 'Казань', 'Томск']
students = [120, 80, 65]

plt.bar(cities, students)
plt.title('Количество студентов')
plt.show()
Bar plot — это сравнение групп. Например, продажи по магазинам, число студентов по городам, средняя зарплата по отделам.

Histogram

  • Показывает распределение числовой переменной.
  • Помогает увидеть, где значений много, где мало, есть ли асимметрия или длинный хвост.
  • Не предназначена для сравнения именованных категорий.
ages = [18, 18, 19, 20, 21, 21, 21, 22, 24, 25, 27, 30]

plt.hist(ages, bins=5)
plt.title('Распределение возраста')
plt.xlabel('Возраст')
plt.ylabel('Частота')
plt.show()
Histogram часто используют для возраста, дохода, длины текста, времени ответа модели, количества слов и других числовых величин.

Scatter plot

  • Нужен для изучения связи двух числовых переменных.
  • Помогает увидеть тренд, кластеры, выбросы.
  • Хорошо подходит для вопросов вида: “связан ли X с Y?”.
height = [160, 165, 170, 175, 180]
weight = [52, 60, 68, 74, 82]

plt.scatter(height, weight)
plt.xlabel('Рост')
plt.ylabel('Вес')
plt.title('Связь роста и веса')
plt.show()
Scatter plot — хороший выбор для роста и веса, часов подготовки и оценки, длины текста и времени обработки.

Boxplot

  • Показывает медиану, квартили и потенциальные выбросы.
  • Удобен для быстрого сравнения распределений между группами.
  • Хорошо подходит, когда нужно не просто среднее, а форма разброса.
scores = [45, 48, 50, 51, 53, 55, 56, 58, 90]

plt.boxplot(scores)
plt.title('Распределение баллов')
plt.show()
Boxplot полезен, если нужно увидеть устойчивый центр распределения, разброс и подозрительные выбросы.

Heatmap

  • Используется для матриц значений.
  • Часто показывает корреляции между признаками.
  • Помогает быстро увидеть сильные и слабые связи по цвету.
import seaborn as sns

corr = df[['age', 'salary', 'score']].corr()
sns.heatmap(corr, annot=True, cmap='coolwarm')
Heatmap особенно полезна для корреляционных матриц, таблиц ошибок, сводных таблиц и пересечений признаков.

Оформление графиков

  • У графика должны быть понятные заголовок и подписи осей.
  • Легенда нужна, если на одном графике несколько серий.
  • Слишком много цветов, подписей и линий перегружают график.
  • Хороший график отвечает на вопрос быстро и без лишнего шума.
plt.figure(figsize=(8, 4))
plt.plot(x, y, label='accuracy')
plt.title('Качество модели по эпохам')
plt.xlabel('Эпоха')
plt.ylabel('Accuracy')
plt.legend()
plt.grid(True)
plt.show()
Всегда спрашивайте себя: что должен понять студент или читатель за 5 секунд просмотра графика?
