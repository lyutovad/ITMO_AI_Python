Зачем Plotly

  • Plotly строит интерактивные графики: можно наводить мышкой, масштабировать, скрывать серии.
  • Он хорошо подходит для дашбордов, разведочного анализа и учебных примеров, где важно “потрогать” график.
  • Часто используется в Jupyter Notebook и в веб-интерфейсах.
import plotly.express as px

fig = px.line(df, x='date', y='sales', title='Продажи по датам')
fig.show()
Если нужен интерактив “из коробки”, Plotly часто удобнее статичных библиотек.

Plotly Express

  • plotly.express — быстрый способ построить график одной командой.
  • Подходит для типовых задач: line, bar, scatter, histogram, box, violin, pie.
  • Удобен, когда данные уже лежат в DataFrame.
import plotly.express as px

fig = px.scatter(
    df,
    x='hours',
    y='score',
    color='group',
    title='Связь часов подготовки и оценки'
)
fig.show()
Plotly Express особенно хорош для быстрого старта и учебных задач.

plotly.graph_objects

  • graph_objects даёт больше контроля над графиком.
  • Там вы вручную добавляете серии, настраиваете оси, легенду и элементы layout.
  • Часто используется, когда простой команды Express уже недостаточно.
import plotly.graph_objects as go

fig = go.Figure()
fig.add_trace(go.Scatter(x=[1,2,3], y=[10,15,13], mode='lines', name='A'))
fig.add_trace(go.Scatter(x=[1,2,3], y=[8,11,16], mode='lines', name='B'))
fig.update_layout(title='Две серии на одном графике')
fig.show()
Если нужен более тонкий контроль, обычно переходят к go.Figure().

Интерактивность

  • В Plotly можно приближать график, выделять участок, читать подсказки при наведении.
  • Легенда часто позволяет скрывать и показывать отдельные серии.
  • hover_data помогает добавить полезные поля во всплывающую подсказку.
fig = px.scatter(
    df,
    x='height',
    y='weight',
    hover_data=['name', 'age']
)
fig.show()
Интерактивность полезна, когда важно исследовать отдельные точки и детали.

Настройка графиков

  • В Plotly часто используют update_layout и update_traces.
  • Можно менять заголовок, подписи осей, легенду, размер графика, шаблон оформления.
  • Для разделения по группам полезны color, symbol, facet_row, facet_col.
fig = px.bar(df, x='city', y='students', color='program')
fig.update_layout(
    title='Студенты по городам',
    xaxis_title='Город',
    yaxis_title='Количество'
)
fig.show()
Хороший Plotly-график — это не только интерактивность, но и понятная структура.

Сохранение графиков

  • Графики Plotly можно сохранять как HTML.
  • Это удобно, если нужно отправить интерактивный график другому человеку.
  • Для статичных картинок используют дополнительные зависимости.
fig.write_html("plot.html")

# Для изображения:
# fig.write_image("plot.png")
HTML — очень удобный формат для обмена интерактивными визуализациями.
