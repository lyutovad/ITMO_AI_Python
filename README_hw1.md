# ITMO AI Python HW1

This repository contains the solution to Homework 1 of the ITMO AI Python course.

## Notebook
The main work is done in `Homework1.ipynb`. Open it with Jupyter and run "Restart & Run All" to execute all cells. The notebook covers:

- string parsing with exception handling
- loops and list functions
- lambda sorting
- a simple `Chocolate` class with validation

## Requirements

A Python 3.8+ environment is assumed. No extra packages are needed.

## Submitting

Create a branch named `hw_1`, add your notebook, commit, and push. Then open a pull request for review.
